linux进程后台执行

发表于 2019-05-18 | 分类于 linux

之前一直没注意的一个小技巧：在终端或者是xshell远程链接服务器时，总会出现关闭终端或连接，上面跑的服务就也会停止的情况，使用nohup命令可以避免。

使用nohup可以不挂断的执行命令（no hang up）

语法：nohup Command [ Arg … ] [　& ]

无论是否将 nohup 命令的输出重定向到终端，输出都将附加到当前目录的 nohup.out 文件中。

最后加上&表示：在后台运行
