渗透的几种方式

att&ck 架构的学习

链接：https://attack.mitre.org

MITRE ATT&CK框架提供了一个较为全面的渗透攻击的技术组成与攻击流程，通过该框架的矩阵视图，可以清晰地看出哪种技术实际上应用于哪个阶段。

ATT&CK矩阵基于国际安全社区对真实世界的观察和研究，收集了200多种攻击者战术和技术。

在该框架中，攻击分为12个阶段：

  • Initial Access
  • Execution
  • Persistence
  • Privilege Escalation
  • Evasion
  • Credential Access
  • Discovery
  • Lateral Movement
  • Collection
  • Command and Control
  • Exfiltration
  • Impact

初始访问：

  • 利用漏洞（如sql注入等）；
  • 利用vpn等有效账户；
  • 利用外部远程服务；
  • 硬件：被动网络窃听，通过DMA读取内核内存，增加新的无线接入现有网络
  • 将恶意软件复制到可移动媒体并在将媒体插入系统并执行时利用自动运行
  • 鱼叉式钓鱼附件；鱼叉式钓鱼链接
  • 供应链
  • 可信关系
