社会工程学攻击

简要描述下社会工程学的攻击方式和攻击工具，在现实中可能会有较大的用处，但是不要轻易向别人发送恶意电子邮件啊…………

近似域名（Doppelganger Domain）

mail.cyberspacekittens.com 和 mailcyberspacekittens.com

快速克隆Web应用程序登录验证页的最佳工具之一是 TrustedSec 公司开发的社会工程学工具包（Social Engineering Toolkit，简称 SET），地址：https://github.com/trustedsec/social-engineer-toolkit 。

解决双因素认证（2FA）

工具：ReelPhish

Microsoft Word/Excel 宏文件

可以使用 Empire 或 Unicorn 来创建一个 VBA 宏

更自动化：LuckyStrike

隐藏加密 payload

EmbededInHTML 和 demiguise

攻击内网

利用未经身份验证的Jenkins应用程序：2.x之前的版本

流程：

  • 受害者访问我们存储的 XSS 或恶意 JavaScript 的页面。
  • 受害者的浏览器执行 JavaScript/WebRTC 以获取内网 IP 并使用 Groovy POST Payload 对本地内部网络发起攻击。
  • 找到一个 Jenkins 服务器后，我们的 Groovy 代码将通知 Jenkins 服务器从攻击者的服务器获取加密的 payload ，然后解密并执行二进制文件。
  • 在这种情况下，我们下载的加密可执行文件是 Meterpreter payload。
  • Meterpreter 在 Jenkins 服务器上执行，然后连接到我们的攻击者 Meterpreter 服务器。
