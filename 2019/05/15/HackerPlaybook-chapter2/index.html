云扫描

首先要知道几大服务提供商的IP地址的范围

主要的搜索引擎：

  • shodan : 定期扫描互联网，抓取banners、端口、网络信息等
    title: 搜索从 HTML 标记中提取的内容
    html: 搜索返回页面的完整 HTML 内容
    product: 搜索 banner 中标识的软件或产品的名称
    net: 搜索一个指定的网段(例如:204.51.94.79/18)
  • Censys.io : 从SSL证书中提取信息，分析该服务器的托管商是谁，查询脚本：https://github.com/christophetd/censys-subdomain-finder

手动解析SSL证书

sslScrape利用 Masscan 快速扫描大型网络。一旦它识别到443端口的服务，它就会在 SSL 证书中提取主机名。

注：有时间学习下sslScrape的源码

子域名收集

Discover Scripts工具:集合了kali上的很多子域名收集工具

KNOCK: 通过一个 wordlist 来枚举目标域中的子域名，严重的依赖字典的质量

Sublist3r:通过搜索引擎,爬虫收集

Truffle Hog,git-all-secrets:从github仓库中提取机密数据、密码、密钥等
